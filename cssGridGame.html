Welcome to Spider Grid, where you write CSS code to catch your prey! Highlight only the areas that have insects by using the grid-column-start property.

For example, grid-column-start: 3; will highlight the area starting at the 3rd vertical grid line, which is another way of saying the 3rd vertical border from the left in the grid.

When grid-column-start is used alone, the grid item by default will span exactly one column. However, you can extend the item across multiple columns by adding the grid-column-end property.

Using grid-column-end, eat all the insects while using the shortest. We don't want to let them run away ! Note that the insects start at the 1st vertical grid line and end at the 4th.

If you want to count grid lines from the right instead of the left, you can give grid-column-start and grid-column-end negative values. For example, you can set it to -1 to specify the first grid line from the right.

Try setting grid-column-end to a negative value.
